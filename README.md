# Cryptojs::Aes::Decrypt

This Gem helps to decrypt crypto-js aes ciphers

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'cryptojs-aes-decrypt'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cryptojs-aes-decrypt
